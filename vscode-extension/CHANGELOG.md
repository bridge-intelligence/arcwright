# Changelog

## 0.1.0 (2026-04-09)

### Features
- TypeScript static analyzer using TS Compiler API
- Interactive React Flow dependency graph in VS Code sidebar
- File watcher with debounced incremental re-analysis
- Dead code detection (unused files and exports)
- Circular dependency detection (Tarjan's DFS)
- `.arcwright.yml` architecture boundary enforcement
- Drift detection with VS Code diagnostics
- Claude Code bridge for context-aware AI prompts
- Arcwright cloud API integration (login, AI analysis)
- Click-to-open: click any node to open the file
- Directory-based color coding
- Stats bar with file count, imports, exports, duration

### Commands
- Show Architecture, Analyze Workspace, Focus Current File
- Login/Logout from Cloud API
- Analyze with AI (Cloud)
- Ask Claude About This File
